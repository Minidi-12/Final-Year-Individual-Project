# Menstrual Health Vulnerability System

AI-powered system for identifying and addressing menstrual health vulnerabilities in Sri Lanka.

## Project Structure

- `data/`: All data files (not committed to git)
- `notebooks/`: Jupyter notebooks for exploration
- `src/`: Source code for models, API, data processing
- `tests/`: Test files
- `docs/`: Documentation
- `outputs/`: Generated figures, models, reports

## Setup

1. Python 3.14
2. PostgreSQL 15+ with PostGIS
3. Install dependencies: `pip install -r requirements.txt`

## Database

- Database name: menstrual_health
- See `src/database/schema.sql` for schema
